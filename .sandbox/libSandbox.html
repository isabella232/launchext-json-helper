Reactor Extension Library Sandbox

How to test:

  1. Open the browser console.
  2. To test the "Parse" data element: run _satellite.getVar('parsed sampleString').
  3. To test the "Stringify" data element: run _satellite.getVar('stringified sampleObject').
  4. To test the null data element: run _satellite.getVar('parsed nullObject').
  5. To test the undefined data element: run _satellite.getVar('stringified undefinedObject').
